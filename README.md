# FileTagManager
ZIPファイルの中の画像を見ながら，ファイル名を変更することができます．

ファイル名は正規表現を用いてタグとして抽出することができます．

# 使用方法
1. フォルダを指定
1. 文字列抽出方法を設定
1. 出力するファイル名のフォーマットを指定

# 設定
XMLファイルに以下の情報が保存されます
* 文字列抽出情報
    * 抽出用正規表現
    * 置換対象の文字列抽出のための正規表現
    * 置換後の文字列
